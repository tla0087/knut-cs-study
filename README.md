# 📝commit 방법

- 파일명 작성 방법 및 파일 추가 방법
  - `jswith`가 `1-3. 운영체제의 구조`를 공부했다고 가정하면,
  - `jswith` branch로 이동한다.
  - `os/jswith/1-3.md`에 공부한 내용을 작성한다.
  - `os: 1-3. 운영체제의 구조` 형태로 commit한다.
  - 원격 저장소에 push한다.
  - `master` branch로 Pull Request를 보낸다.
