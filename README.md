# 방식

- 월 ~ 금 하루 1시간 씩만, 주말은 휴식!
- 매주 한 개의 챕터 선정 -> 하루에 소분류 하나씩
- 스스로 공부하고 이해한 내용 각자 폴더에 업로드
- 화상(Google Meet)으로 상호 질문 및 점검, 틀린 부분 보완
